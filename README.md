# MacroRecorder

A powerful keyboard and mouse macro recording and playback application for people who want to AFK in games.

**Created by: Nölli_Hestur**

---

## Installation

1. **Extract the ZIP file** to any folder
2. **Double-click "Install to Desktop.bat"** - this will create a shortcut on your Desktop
3. **Done!** You can now launch MacroRecorder from your Desktop

The app stores all your profiles in a `profiles` folder created automatically when you first run it.

---

## Quick Start Guide

### Recording a Macro

1. Go to the **Recorder** tab
2. Click **RECORD** button (or press **F6** by default)
3. Perform your actions (clicks, key presses, etc)
4. Click **STOP** button (or press **F9** by default) to finish recording
5. You'll see the number of recorded events

### Playing Back a Macro

1. Click **PLAY MACRO** button (or press **F10** by default)
2. Adjust **Speed** (0.25x to 4.0x) to make it faster or slower
3. Set **Repeat** to run the macro multiple times, or check "Loop forever" for unlimited repeats
4. Your macro will playback automatically

### Saving Profiles (Presets)

1. Record a macro or load an existing one
2. Go to the **Profiles** tab
3. Enter a **Profile name** (e.g., "AFK Grind", "Boss Fight")
4. Click **Save as Profile**
5. Your profile is saved and appears in the profile list

### Loading Profiles

1. Go to the **Profiles** tab
2. Click on a profile name in the list to see details
3. Click **Load** to load it into the recorder
4. You can now play it back or edit it

### Autoclicker

1. Go to the **Autoclicker** tab
2. Set the **Click delay** (in milliseconds)
3. Choose your **Mouse button** (left, right, or middle)
4. Click **START AUTOCLICKER** (or press **F9** by default to stop)
5. The panic button (🛑) will emergency stop everything

## Recording Options

- **Record Keyboard**: Toggle keyboard input recording
- **Record Mouse**: Toggle mouse movement and clicks recording

Check at least one option before recording!

---

## Tips & Tricks

1. **Adjust playback speed** if macros are too fast or slow
2. **Use profiles** to organize different macros (one per game/task)
3. **Toggle recording preview** with "🔍 Show Recording Preview" to see what's being recorded
4. **Use the panic button** (🛑) for instant emergency stop
5. **Customize hotkeys** to match your preferences
6. **Clear preview** between recordings to keep things clean

## Troubleshooting

**Issue: Hotkeys not working?**
- Make sure the app window is active (or minimized but the app is running)
- Try rebinding the hotkey - click the button and press a new key

**Issue: Playback seems janky?**
- Adjust the **Speed** slider - try 1.0x first
- Reduce other running applications to free up CPU

**Issue: Mouse clicks not recording?**
- Check the **Record Mouse** checkbox under Recording Options

## Keyboard Shortcuts During Recording

- Press the **STOP hotkey** (F9 default) to stop recording
- Press the **PANIC hotkey** (F8 default) to emergency shutdown
- Press the **PLAY hotkey** (F10 default) to play the recording
